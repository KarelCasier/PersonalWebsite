Faux 3D Renderer
Project Summary

A test using vertex arrays to fake 3d perspective of buildings.
Code available to view on Github

Tech Used

C++ | SFML

Details

All rendered in 2D using vertex arrays to draw the building, the perspective comes from manipulating the verticies of the vertex array for each object based on camera position, object height and z-position. The objects are then sorted by distance from camera location before rendering to ensure closer objects are drawn over further objects.

Each object transforms it's own verticies so the engine is quite easy to add new objects. Simply creating a new object with a position, dimentions, z position(how high off the ground) and which texture to use is all that is needed.

The example below shows a simple city containing a few different objects: buildings, building tunnels, lamp posts and sign panels.
