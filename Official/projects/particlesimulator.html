Particle Simulator
Project Summary

A particle simulator that is capable of handling up to 250,000 particles at once smoothly. This was a mini project written in 2 days.
Code available to view on Github

Tech Used

C++ | SFML

Windows Download
OSX Download
Details

A vertex array is used to draw the particles using a class similar to gl_points.

You can directly interact with the particles using the mouse (either attract or repel) or indirectly by placing effectors (gravity well or repulsor)

Controls
Key Action
M Change Mode
Left Click ( In Direct Mode ) Attract Particles
Right Click ( In Direct Mode ) Repel Particles
Left Click ( In Indirect Mode ) Add Gravity Well
Right Click ( In Indirect Mode ) Add Repulsor
C Clear Effectors
R Reset Particles
W Toggle Particle Screen Wrapping
D Toggle Particle Drag
Right Arrow Increase Particle Count
Left Arrow Decrease Particle Count
