UNTITLED SPACE GAME
Work in progress
Project Summary

"Untitled Space Game" is a 2D procedurally generated space game.
Code available to view on Github

Tech Used

C++ | SDL

Details

The game is programmed using Object Oriented Design patterns such as inheritance, polymorphism and singletons.

At this point, the game is still in heavy development so there is not too much to show yet as I am working on the code base of the project which includes: game state machine, object and texture manager, particle system, layer renders that use custom cameras to offset the the render area (so that the 'camera' can follow and move).

Something that I just recently finished was the natural looking star layouts. I did this using a form of noise called simplex noise that produces gradual gradients that can be infinite in size. Using this data I generate a spot for a star then check if it 'survives' by checking the value generated at that position by the noise.
